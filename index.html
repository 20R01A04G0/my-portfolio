Room Air Purifier

An Internet of Things (IoT) gadget called a room air purifier is composed of both hardware and software.I served as the team leader for this project that we created together.This tool essentially counts the number of contaminants in a space. We utilised an LCD display in this to show the quantity of pollutants, and the programming was built in Arduino.If the level of pollutants rises to 200 ppm, we turn on the purifier, which is built with standard parts like bike and automobile filters.Comparatively speaking, it is a cheap air purifier.

Temperature conversion

The temperature conversion interface is designed using HTML, CSS, and JavaScript. It features two input fields for Celsius and Fahrenheit temperatures, allowing users to enter values to convert. Upon clicking the "Convert" button, the JavaScript code calculates and displays the converted temperatures in the respective result fields. The design is clean and user-friendly, with a centered heading, labeled input fields, and a visually appealing button. The result section provides clear labels for the converted temperatures.The CSS styling enhances the interface's readability and visual appeal, with consistent spacing, font styles, and a light background color.
